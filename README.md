# php-chunkifier

## Quick Start
- Install dependencies ``` composer install ```
- To chunk file run : ``` php run.php data/data.txt /home/chunk/ ```

## Test
- Run the following command to test ``` ./vendor/bin/phpunit ./tests/ ```
- Go to ``` /tests/ ``` to see test files
